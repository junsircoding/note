将信息按照每条长度不大于1000拆分

```python
msg = "xx*xx"
msg_cut = msg.split("*", -1)
# 将信息按照每条长度不大于1000拆分
msg_list = []
temp_msg = ""
for index in range(len(msg_cut)):
    if len(temp_msg) + len(msg_cut[index]) < 1000:
        temp_msg = temp_msg + msg_cut[index]
    else:
        msg_list.append(temp_msg)
        temp_msg = ""
        index = index - 1
```
